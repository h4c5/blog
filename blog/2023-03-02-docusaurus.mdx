J'ai décidé de créer ce blog sous forme de gros bloc-notes publique où je pourrais noter les choses que
j'apprends au quotidien.

Le code et les notes sont accessibles sur github afin que tout le monde puisse proposer des améliorations.

J'utilise [Docusaurus](https://docusaurus.io/) qui est un générateur de site statique pensé pour la documentation de
projet informatique. Cela permet d'écrire facilement des articles et des posts au format markdownx, c'est-à-dire avec la
possibilité d'inclure des composants React comme celui-ci :

<ExampleComponent />

Je ne m'y connais pas trop en React mais ça m'a l'air pas mal du tout. J'ai par exemple ajouté la section commentaire
grâce à un composant React [Gisus](https://giscus.app/fr).
