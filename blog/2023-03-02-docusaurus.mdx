J'ai décidé de créer ce blog sous forme de gros bloc-notes publique où je pourrais noter les choses que
j'apprends au quotidien.

L'idée est que le code et les notes et articles (écrits en markdown mdx) soient accessibles sur github afin que tout le
monde puisse proposer des améliorations.

J'utilise [Docusaurus](https://docusaurus.io/), le générateur de site statique pensé pour la documentation de projet
informatique. Cela permet d'écrire facilement des articles et des posts au format markdownx, c'est-à-dire avec la
possibilité d'inclure des composants React :

<ExampleComponent />

Je ne m'y connais pas trop en React mais ça m'a l'air pas mal du tout. J'ai par exemple ajouté la section de
commentaires grâce à un composant React [Gisus](https://giscus.app/fr).
